# Challenge-Foro

This is a RESTful API developed as part of the Oracle Next Education backend specialization.  The goal of this project is to manage a forum's data, allowing users to interact with topics and answers through a secure interface.

Features
* Full CRUD Operations: Create, Read, Update, and Delete for topics and users.
* Token-based Authentication: Secure access using JWT to protect sensitive endpoints.
* Database Integration: Persistent data management using MySQL/PostgreSQL.
* JSON Processing: Seamless communication between client and server using JSON format.

Tech Stack
* Language: Java
* Framework: Spring Boot
* Tools: Git, GitHub, Postman (for API testing)
* Database: MySQL

How to run it
1. Clone this repository.
2. Configure your database credentials in application.properties.
3. Run the application using your favorite IDE (IntelliJ, Eclipse, etc.).
4. Test the endpoints using the provided Postman collection.
